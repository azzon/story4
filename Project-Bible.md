## 二、市场研究摘要

### 2.1 平台生态结论

经过对起点中文网、番茄小说、七猫免费小说三大平台的深度调研：

**平台选择建议**: 起点中文网
- 收入上限最高，头部作者年入千万级
- 读者付费意愿强，忠诚度高
- IP衍生价值最大，改编机会多
- 适合400万字长篇精品路线

### 2.2 品类趋势

**主流品类**:
- 玄幻/仙侠：占比26.7%，市场最大但竞争最激烈
- 都市/现言：占比24.0%，稳定产出爆款
- 甜宠/霸总：占比18.7%，主要在番茄/七猫

**新兴品类**（上升趋势）:
1. 无限流/规则怪谈 - 硬核读者喜爱
2. 直播/短视频流 - 契合时代热点
3. 模拟经营/种田升级版 - 养成爽感

### 2.3 标杆作品

| 作品 | 类型 | 核心成功要素 |
|------|------|--------------|
| 《诡秘之主》 | 克苏鲁+蒸汽朋克 | 独创力量体系、精密伏笔网络 |
| 《斗破苍穹》 | 东方玄幻 | 废物流模板、频繁打脸、地图扩展 |
| 《全职高手》 | 电竞/游戏 | 大神归来、群像塑造、热血竞技 |
| 《凡人修仙传》 | 仙侠修真 | 凡人流开创、谨慎型主角、史诗级架构 |
| 《我有一座恐怖屋》 | 灵异悬疑 | 副本+经营双线、恐怖氛围、群像鬼怪 |

### 2.4 市场机会缺口

**过度饱和的元素**:
1. 系统/面板流 - 同质化严重，读者审美疲劳
2. 无限流/副本流 - 模仿者众多但缺乏创新
3. 赘婿/废物开局 - 套路模板化，缺乏突破

**供给不足的方向**:
1. 中国神话体系深度开发 - 评分8.5/10
2. 中年/成熟向市场 - 评分8.0/10
3. 轻量化史诗（300万字精品） - 评分7.5/10

**创新组合机会**:
1. 凡人流+灵异悬疑 - 评分8.0/10
2. 鬼屋经营+都市职场 - 评分8.0/10
3. 电竞+仙侠修真 - 评分7.0/10

## 三、核心概念（Stage 01 创意孵化进行中）

### 3.1 方案A - 东方玄幻/神话复苏

#### 一句话卖点
> 落魄道士血脉觉醒，借封神榜规则收服山海异兽，重建人间神庭。

#### 主角设定
- **姓名**：陆青玄，27岁
- **身份**：正一派末代传人，殡葬司仪
- **性格**：务实隐忍、多疑冷漠、狠辣偏执
- **核心欲望**：掌控感
- **致命弱点**：无法接受被背叛

#### 外挂/金手指
- **名称**：封神榜碎片·人榜残卷
- **核心能力**：敕封异兽、神位升格、神念共鸣、借神威
- **代价**：寿元消耗、神魂反噬、天道注视

- **对手**：天庭司命府
- **终极BOSS**：文昌帝君
- **冲突原因**：封神榜觉醒引来天庭追杀 + 师门覆灭复仇

**胜出方案**: 方案A · 东方玄幻/神话复苏

**评估得分**: 7.8/10（三方案最高）

| 维度 | 得分 |
|------|------|
| 商业潜力 | 8/10 |
| 可扩展性 | 9/10 |
| 差异化程度 | 8/10 |
| 执行难度 | 6/10 |
| 个人兴趣/优势 | 8/10 |

**决策理由**:
1. 市场时机最佳——国风崛起的时代红利
2. 可扩展性最强——450+异兽+365位正神的素材库
3. 差异化与可读性平衡——创新但不曲高和寡
4. 风险可控——架空设定+成熟框架
5. IP价值最大——动画、游戏、周边全面开花

**一句话卖点（锁定版）**:

## 六、金手指设定 · 封神榜碎片

### 6.1 来历与获得

封神榜原为上古神器，封神之战后分为三块碎片：天榜（天庭）、地榜（地府）、人榜（正一派）。陆青玄三岁濒死时，师父以人榜残卷救命，碎片融入眉心。二十四年后在食人魈袭击中觉醒。

### 6.2 核心能力

| 能力 | 效果 | 限制 |
|------|------|------|
| **敕封** | 将异兽神魂纳入封神榜，成为"神将" | 需对方臣服，耗时1-3时辰 |
| **神位升格** | 消耗材料提升神将力量 | 有失败风险，可能暴走 |
| **神念共鸣** | 与神将共享感官 | 本体意识变迟钝 |
| **借神威** | 临时借用神将力量附身 | 时间有限，事后虚弱 |

### 6.3 使用代价

- **寿元消耗**：高级能力消耗寿元（敕封半年~三年不等）
- **神魂反噬**：神将过多会干扰意识（噩梦→人格分裂）
- **天道注视**：频繁使用暴露风险增加

### 6.4 成长路径

| 境界 | 状态 | 上限 |
|------|------|------|
| 练气期 | 残破 | 5位 |
| 筑基期 | 初步修复 | 12位 |
| 金丹期 | 基本修复 | 36位 |
| 元婴期 | 完全修复 | 72位 |
| 化神期+ | 解锁隐藏功能 | 365位 |

### 6.5 天花板设定

- 数量上限：365位正神（人榜容量）
- 等级上限：只能敕封"人间级"存在，不可敕封天庭现有神明
- 质量限制：异兽实力受限于陆青玄自身境界
