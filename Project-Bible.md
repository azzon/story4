## 二、市场研究摘要

### 2.1 平台生态结论

经过对起点中文网、番茄小说、七猫免费小说三大平台的深度调研：

**平台选择建议**: 起点中文网
- 收入上限最高，头部作者年入千万级
- 读者付费意愿强，忠诚度高
- IP衍生价值最大，改编机会多
- 适合400万字长篇精品路线

### 2.2 品类趋势

**主流品类**:
- 玄幻/仙侠：占比26.7%，市场最大但竞争最激烈
- 都市/现言：占比24.0%，稳定产出爆款
- 甜宠/霸总：占比18.7%，主要在番茄/七猫

**新兴品类**（上升趋势）:
1. 无限流/规则怪谈 - 硬核读者喜爱
2. 直播/短视频流 - 契合时代热点
3. 模拟经营/种田升级版 - 养成爽感

### 2.3 标杆作品

| 作品 | 类型 | 核心成功要素 |
|------|------|--------------|
| 《诡秘之主》 | 克苏鲁+蒸汽朋克 | 独创力量体系、精密伏笔网络 |
| 《斗破苍穹》 | 东方玄幻 | 废物流模板、频繁打脸、地图扩展 |
| 《全职高手》 | 电竞/游戏 | 大神归来、群像塑造、热血竞技 |
| 《凡人修仙传》 | 仙侠修真 | 凡人流开创、谨慎型主角、史诗级架构 |
| 《我有一座恐怖屋》 | 灵异悬疑 | 副本+经营双线、恐怖氛围、群像鬼怪 |

### 2.4 市场机会缺口

**过度饱和的元素**:
1. 系统/面板流 - 同质化严重，读者审美疲劳
2. 无限流/副本流 - 模仿者众多但缺乏创新
3. 赘婿/废物开局 - 套路模板化，缺乏突破

**供给不足的方向**:
1. 中国神话体系深度开发 - 评分8.5/10
2. 中年/成熟向市场 - 评分8.0/10
3. 轻量化史诗（300万字精品） - 评分7.5/10

**创新组合机会**:
1. 凡人流+灵异悬疑 - 评分8.0/10
2. 鬼屋经营+都市职场 - 评分8.0/10
3. 电竞+仙侠修真 - 评分7.0/10

## 三、核心概念（Stage 01 创意孵化进行中）

### 3.1 方案A - 东方玄幻/神话复苏

#### 一句话卖点
> 落魄道士血脉觉醒，借封神榜规则收服山海异兽，重建人间神庭。

#### 主角设定
- **姓名**：陆青玄，27岁
- **身份**：正一派末代传人，殡葬司仪
- **性格**：务实隐忍、多疑冷漠、狠辣偏执
- **核心欲望**：掌控感
- **致命弱点**：无法接受被背叛

#### 外挂/金手指
- **名称**：封神榜碎片·人榜残卷
- **核心能力**：敕封异兽、神位升格、神念共鸣、借神威
- **代价**：寿元消耗、神魂反噬、天道注视
- **上限**：最多敕封365位异兽

#### 核心冲突
- **对手**：天庭司命府
- **终极BOSS**：文昌帝君
- **冲突原因**：封神榜觉醒引来天庭追杀 + 师门覆灭复仇

---

### 3.2 方案B - 都市灵异/鬼屋经营

#### 一句话卖点
> 996社畜继承诡异游乐园，解锁恐怖副本收服鬼怪，打造最强阴间娱乐帝国。

#### 主角设定
- **姓名**：沈默，26岁
- **身份**：前互联网产品经理，阴乐园第三代园主
- **性格**：冷静抠门、社恐记仇、敬业麻木
- **核心欲望**：财务自由
- **致命弱点**：无法拒绝合理的交易

#### 外挂/金手指
- **名称**：阴乐园APP
- **核心能力**：副本探索、设施升级、鬼怪雇佣、任务系统
- **代价**：精神消耗、债务压力、员工管理
- **上限**：游乐园等级与APP功能挂钩

#### 核心冲突
- **对手**：夜巡局
- **终极BOSS**：副局长周尔仁
- **冲突原因**：游乐园接管权争夺 + 外公死亡真相

### 3.3 方案C - 历史架空/战国策士

#### 一句话卖点
> 落魄书生魂穿战国，凭博弈残篇纵横七国，从食客到天下第一谋士。

#### 主角设定
- **姓名**：卫鞅，25岁
- **身份**：前世985博弈论研究生，魂穿战国成为落魄门客
- **性格**：冷静傲慢、坚韧偏执、善谋薄情
- **核心欲望**：证明自己——智力上的绝对优越
- **致命弱点**：无法接受失败

#### 外挂/金手指
- **名称**：博弈残篇
- **核心能力**：局势推演、纳什预言、机制设计、信息操控
- **代价**：头痛反噬、记忆消耗、信息依赖
- **上限**：非理性对手、绝对武力差距、时间限制

#### 核心冲突
- **对手**：公孙鞅（另一个穿越者，历史系教授）
- **冲突本质**：历史vs博弈——命运是否可以改变？
- **终极对决**：秦国变法执行者之争

### 3.4 最终选定方案

**胜出方案**: 方案A · 东方玄幻/神话复苏

**评估得分**: 7.8/10（三方案最高）

| 维度 | 得分 |
|------|------|
| 商业潜力 | 8/10 |
| 可扩展性 | 9/10 |
| 差异化程度 | 8/10 |
| 执行难度 | 6/10 |
| 个人兴趣/优势 | 8/10 |

**决策理由**:
1. 市场时机最佳——国风崛起的时代红利
2. 可扩展性最强——450+异兽+365位正神的素材库
3. 差异化与可读性平衡——创新但不曲高和寡
4. 风险可控——架空设定+成熟框架
5. IP价值最大——动画、游戏、周边全面开花

**一句话卖点（锁定版）**:
> 末代道士手持封神残卷，收服山海异兽，敢与天庭争人间神位。（28字）
